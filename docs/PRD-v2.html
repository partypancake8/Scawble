Type & shape

font
Fredoka (rounded geometric sans) embedded as woff2 data-URI — weights 400/600. Replaces the book serif.
tiles
border-radius ~26%, big rounded letter, tiny corner value, one soft shadow + lip. Board cells rounded ~22%.
board
rounded card (18px), soft shadow, gentle cell separation — no grey gridlines.
buttons
rounded pills, coral primary, springy press (scale .96).
motion
bouncier settle, cuter score pop, tuned confetti; all respect reduced-motion.

02

P1 features

All seven. Each specified by behavior + an acceptance check. Logic reuses the tested controller.

#7 · you asked

Last move, per player

Under each score, a live line: QUARTZ +32 / AN +2 (or "passed" / "swapped"). Updates every turn from state.history.

Accept After each turn, both lines reflect that side's most recent action + points.
#8

Move log

A toggle opens a scrollable running history of every turn (both players): word/action + score, newest first.

Accept Log entries match state.history in order and count.
#9 · a11y

Blank-tile picker

Replaces the raw prompt(). Dropping a blank opens a rounded A–Z chooser overlay; pick assigns the letter. Keyboard + tap.

Accept A blank can be assigned any A–Z, scores 0, and shows its chosen letter.
#10

Hint

A Hint button calls playerBest() and shows the suggested word as ghost tiles on the board + "Try QUARTZ (+32)". Clears on next action.

Accept Ghost matches the true best legal move for the position.
#11 · a11y/mobile

Tap-to-place

Tap a rack tile to select it, tap an empty cell to place it — alongside drag. Tap a placed draft tile to recall it.

Accept A full word can be built with taps only, no dragging.
#12

Settings screen

Toggles for sound, haptics, reduced-motion (auto/on/off), theme (auto/light/dark), default difficulty. Persist to localStorage; apply immediately.

Accept Each toggle persists across reload and takes effect at once.
#13

Swap selected tiles

Swap enters a select mode: tap rack tiles to mark, confirm to exchange just those (uses the turn). Replaces the "swap whole rack" stub.

Accept Only the chosen tiles are exchanged; illegal when the bag has < that many tiles.
